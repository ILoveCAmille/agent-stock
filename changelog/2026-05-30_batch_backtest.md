## 变更概述

创建150只股票批量回测系统（小市值50只/中市值50只/大市值50只），分别用对应的市值风格权重进行回测。

## 新增文件

### run_batch_backtest.py（528行）
批量回测主脚本，功能包括：
- 预定义150只A股股票池（50+50+50，覆盖创业板/科创板/主板/银行/券商等）
- 按市值分类自动选择对应策略和权重
- 每只股票独立回测并输出收益/夏普/回撤/胜率
- 跨类别对比汇总分析
- 结果保存为JSON和TXT报告

### test_data_api.py
测试8种免费股票数据接口的可用性。

### test_proxy.py
诊断网络代理和连接问题。

## 使用方法

```bash
# 每类10只快速验证
python run_batch_backtest.py --max-stocks 10

# 每类50只全量回测
python run_batch_backtest.py --max-stocks 50

# 只测小市值
python run_batch_backtest.py --cap small_cap --max-stocks 50

# 使用在线数据
python run_batch_backtest.py --online --max-stocks 50
```
