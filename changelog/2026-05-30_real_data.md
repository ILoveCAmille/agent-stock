## 变更概述

解决网络环境下获取真实股票数据的问题，集成多个HTTP数据源，使用新浪K线API实现真实A股数据回测。

## 数据接口测试结果

| 接口 | 类型 | 状态 | 协议 |
|------|------|:---:|------|
| 新浪K线API | 历史K线(最多1023条) | ✅ 稳定 | HTTP |
| 腾讯行情 | 实时行情 | ✅ 稳定 | HTTP |
| 新浪行情 | 实时行情 | ✅ 稳定 | HTTP |
| biyingapi | 实时+基本面(市值/PE/PB) | ✅ 稳定 | HTTP |
| 东方财富HTTP | 历史K线 | ⚠️ 间歇性 | HTTP |
| akshare | 全量数据 | ❌ HTTPS被阻断 | HTTPS |
| 网易财经 | 历史数据 | ❌ 502 | HTTP |
| baostock | 免费A股数据 | ⚠️ 未安装 | - |

## 关键发现

1. **新浪财经K线API** 是最稳定的数据源，HTTP协议不受HTTPS代理影响
2. 接口地址: `http://money.finance.sina.com.cn/quotes_service/api/json_v2.php/CN_MarketData.getKLineData`
3. 每次可获取最多1023条日K线（约4年数据），返回JSON格式
4. 真实A股前复权数据，包含OHLCV

## 新增文件

### real_data_fetcher.py（254行）
真实股票数据获取器，多源自动切换：
- `_fetch_kline_sina()`: 新浪K线（首选，稳定）
- `_fetch_kline_eastmoney()`: 东方财富K线（备选）
- `fetch_realtime()`: 腾讯实时行情
- `fetch_stock_info()`: biyingapi基本面数据
- `batch_fetch_klines()`: 批量获取K线

### fix_network.py
网络诊断工具，检测Windows系统代理设置。

### diagnose_dns.py
DNS解析和各域名连通性诊断工具。
