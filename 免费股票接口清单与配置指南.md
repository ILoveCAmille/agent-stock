## 🚀 推荐新增数据源配置

### 1. 新浪实时行情接口 (推荐)
新浪财经接口是目前速度最快的免费实时行情源，延迟<500ms，无限调用。

可以新增到 `data_source_manager.py` 作为第三备用数据源：

```python
def get_sina_realtime(self, code):
    """新浪实时行情接口"""
    import requests
    url = f"http://hq.sinajs.cn/list=sh{code}" if code.startswith('6') else f"http://hq.sinajs.cn/list=sz{code}"
    headers = {'Referer': 'https://finance.sina.com.cn'}
    resp = requests.get(url, headers=headers)
    data = resp.text.split('"')[1].split(',')
    return {
        'name': data[0],
        'price': float(data[3]),
        'pre_close': float(data[2]),
        'open': float(data[1]),
        'high': float(data[4]),
        'low': float(data[5]),
        'volume': int(data[8]),
        'amount': float(data[9])
    }
```

def get_qq_realtime(self, code):
