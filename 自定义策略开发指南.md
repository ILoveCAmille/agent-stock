# 自定义股票策略开发完整指南

## 🚀 3分钟快速上手

### 第一步: 复制模板文件
```bash
# 复制选股模块
copy custom_strategy_template.py my_strategy_selector.py

# 复制UI模块 (参考现有策略)
copy low_price_bull_ui.py my_strategy_ui.py
```

### 第二步: 修改你的策略逻辑
编辑 `my_strategy_selector.py`
1. 修改类名 `CustomStrategySelector` → `MyStrategySelector`
2. 修改 `STRATEGY_NAME` 和 `STRATEGY_DESC`
3. 在 `CONDITIONS` 字典中定义你的选股条件
4. 或者完全重写 `get_stocks()` 方法实现你的逻辑

### 第三步: 注册到主界面
在 `app.py` 中添加3处修改:

#### 1. 顶部导入
```python
# 在其他策略导入旁边添加
from my_strategy_ui import display_my_strategy
```

#### 2. 侧边栏菜单
在侧边栏的"📊 选股策略"区域添加:
```python
if st.sidebar.button("🎯 我的自定义策略", use_container_width=True):
    st.session_state.page = "my_strategy"
```

#### 3. 路由判断
在页面路由判断区域添加:
```python
elif st.session_state.page == "my_strategy":
    display_my_strategy()
```

### 第四步: 测试运行
```bash
python run.py
```
打开浏览器即可看到你的策略出现在侧边栏菜单中！

## 💡 开发最佳实践

### 选股方式选择
| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **问财接口** | 简单、强大、支持自然语言 | 依赖外部接口 | 90%的常规策略 |
| **AKShare本地计算** | 完全可控、无依赖 | 代码量大、速度慢 | 特殊自定义指标 |
| **Tushare** | 数据稳定 | 需要Token | 专业财务数据 |

### 代码规范
1.  所有策略文件遵循统一命名规范: `策略名_selector.py` `策略名_ui.py`
2.  选股类必须实现 `get_stocks()` 方法，返回标准的 (success, df, msg) 格式
3.  必须实现 `get_clean_codes()` 方法返回纯净股票代码列表
4.  UI层复用系统提供的通用组件，不要自己重写表格、按钮等

## ⚠️ 注意事项

1.  **选股数量控制**: 单次选股建议不要超过20只，否则AI分析会很慢
2.  **API限流**: 问财接口有调用频率限制，不要短时间频繁调用
3.  **异常处理**: 所有网络请求必须加异常捕获
4.  **日志输出**: 重要步骤必须有控制台日志，便于调试
5.  **数据验证**: 选股结果必须做非空和类型检查
